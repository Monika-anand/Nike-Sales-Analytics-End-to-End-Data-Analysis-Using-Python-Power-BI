# Nike-Sales-Analytics-End-to-End-Data-Analysis-Using-Python-Power-BI
This project performs end-to-end data analysis on sales data of Nike using Python and Power BI. The goal is to clean raw data, analyze sales performance, and build an interactive dashboard for business insight


## 🔍 Project Overview
This project focuses on analyzing sales data of Nike using Exploratory Data Analysis (EDA) and Power BI dashboards. The objective is to uncover insights related to revenue trends, product performance, regional demand, and profitability.

The project transforms raw and uncleaned sales data into meaningful business insights to support data-driven decision-making.

---

## 🎯 Project Objective
- Analyze sales performance across regions, products, and channels  
- Identify key revenue drivers and profitable segments  
- Understand the impact of discounts on profitability  
- Compare online and retail sales performance  
- Build an interactive dashboard for business insights  

---

## 🛠 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn) – Data Cleaning & EDA  
- Power BI – Dashboard & Visualization  
- Microsoft Excel – Data Handling

## 📊 Key Metrics
- **Total Revenue:** $25.16M  
- **Total Profit:** $4.10M  
- **Profit Margin:** 16.31%  
- **Units Sold:** 4,755  
- **Average Discount:** 13.01%  

---

## 📈 Key Insights
- Training and Lifestyle categories generate the highest revenue  
- Online and Retail channels contribute equally (~50%)  
- Metro cities (Delhi, Mumbai, Bangalore) dominate sales  
- Revenue shows consistent growth from 2023 to 2025  
- Moderate discounts improve sales, but high discounts reduce profit  
- A few top products contribute most of the revenue (Pareto Principle)
